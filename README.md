# Проекты

## 🚀 Explore With Me

Микросервисная платформа для организации и поиска мероприятий.

### Что реализовано

* микросервисная архитектура (основной сервис + сервис статистики)
* три уровня API: публичный, пользовательский и административный
* работа с PostgreSQL через Hibernate (JPA)
* контейнеризация сервисов через Docker
* сбор статистики просмотров мероприятий
* тестирование API через Postman

### Технологии

Java 17, Spring Boot, Hibernate, PostgreSQL, Docker, Maven, Postman, Microservices

## 🏦 Bank REST API

REST API банковского сервиса с JWT-аутентификацией и ролевой моделью доступа.

### Что реализовано

* аутентификация и авторизация через JWT
* роли USER и ADMIN
* управление пользователями и банковскими картами
* переводы между картами
* централизованная обработка ошибок
* миграции базы данных через Liquibase
* документация Swagger/OpenAPI
* Docker Compose развёртывание

### Технологии

Java 17, Spring Boot, Spring Security, JWT, PostgreSQL, Hibernate, Liquibase, Docker, Maven, Swagger/OpenAPI
